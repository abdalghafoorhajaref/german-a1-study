# 🇩🇪 تطبيق دراسة اللغة الألمانية A1 (Berliner Platz 1 Neu Study Companion)

تطبيق ويب تفاعلي متكامل لدراسة منهج اللغة الألمانية للمستوى **A1** بناءً على كتاب **Berliner Platz 1 Neu**. التطبيق مصمم ليعمل بالكامل دون الحاجة لخادم (Offline-first / Static Web App)، ويقدم تجربة تعليمية متميزة تدعم اللغتين العربية والألمانية.

An interactive static web application designed to study the German **A1** curriculum based on the **Berliner Platz 1 Neu** textbook. The app runs completely client-side and is optimized for both desktop and mobile viewports.

---

## ✨ المميزات الرئيسية / Key Features

1. **📖 الوحدات والدروس (Lektionen & Kapitel)**
   * تغطية شاملة لجميع وحدات الكتاب الـ 12 بالتفصيل.
   * نصوص القراءة والاستماع الخاصة بكل درس مترجمة للغة العربية.
   
2. **🗣️ محاكي المحادثة التفاعلي (Interactive Conversation Simulator)**
   * 12 سيناريو محادثة تفاعلي يغطي مواقف حقيقية (التعريف بالنفس، السوبرماركت، وصف الطريق، عند الطبيب، إلخ).
   * الاستماع لنطق الطرف الآخر باستخدام تقنية تحويل النص إلى كلام (TTS).
   * ممارسة التحدث عبر نطق الإجابات بالمايكروفون وتقييم جودة النطق بدقة (Speech Recognition) مع إمكانية التخطي اليدوي.

3. **🎧 مشغل الصوتيات (Integrated Audio Player)**
   * تم دمج وتصنيف جميع المقاطع الصوتية (CD1 & CD2) من الكتاب وتسميتها باللغتين العربية والألمانية لسهولة الوصول والاستماع أثناء القراءة.

4. **📝 التدريب على الكتابة (Schreibübungen)**
   * تدريبات كتابية لتعلم كيفية كتابة المواضيع والرسائل والبطاقات البريدية المطلوبة في امتحان A1 مع تصحيح تلقائي وعرض النماذج الصحيحة.

5. **📚 المفردات والقواعد (Wortschatz & Grammatik)**
   * قوائم مفردات تفاعلية لكل وحدة مع النطق والترجمة العربية.
   * شرح مبسط للقواعد الخاصة بكل درس مدعومة بالأمثلة.

6. **🃏 بطاقات التعليم (Flashcards)**
   * نظام بطاقات تعليمية ذكي لمراجعة وحفظ الكلمات والعبارات الهامة.

7. **🏆 الاختبارات (Quizzes)**
   * اختبار تفاعلي لكل وحدة لتقييم مدى فهمك للمفردات والقواعد والاستماع مع إظهار النتيجة النهائية.

## 🚀 طريقة التشغيل المحلي / Local Execution

1. قم بتحميل المشروع أو عمل Clone له.
2. افتح ملف `index.html` في متصفحك مباشرة (Double click) وسيفتح التطبيق فوراً دون الحاجة لتثبيت أي برامج أو خوادم.

---

## 🌐 الاستضافة على GitHub Pages / Hosting on GitHub Pages

للدراسة من التطبيق أونلاين من أي جهاز أو هاتف:
1. ارفع المشروع إلى مستودع (Repository) في حسابك على GitHub.
2. اذهب إلى إعدادات المستودع (**Settings**).
3. اختر قسم **Pages** من القائمة الجانبية.
4. تحت قسم **Build and deployment**، اختر الفرع `main` (أو `master`) واضغط **Save**.
5. سيظهر لك رابط مخصص أونلاين (مثل: `https://username.github.io/repo-name/`) لتتمكن من تصفح واستخدام التطبيق بأكمله!
